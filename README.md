# Estimating wireless backhaul reliability under rain attenuation

Authors: Ufuk Usubutun & Andre Gomes

This repo contains:
* our script used to generate the figures in the paper,
* and an AI agent generated notebook based on our script for easy testing of different parameters

All results in the publication were produced by our own script, which we verified for correctness in the cases considered.

AI-generated results were not used in the publication, and the notebook is only provided as a convenience to the reader. We also did not verify the validity of the results generated by that notebook. Use it at your own risk.
